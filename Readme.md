# ESTUDO COMPLEMENTAR

## Descrição

Projeto pessoal de desenvolvimento técnico criado em paralelo à graduação em Ciência da Computação.

## Objetivo

O foco é fortalecer os fundamentos da programação por meio de estudo contínuo, prática estruturada e documentação da evolução ao longo do tempo, além de criar material de consulta.

As áreas trabalhadas incluem raciocínio lógico, pensamento computacional, análise de problemas, organização de código e domínio das ferramentas essenciais da área.

Este projeto também funciona como preparação estratégica para processos seletivos técnicos, com foco futuro em oportunidades nas áreas de Dados e Inteligência Artificial em empresas como a IBM.

## Tecnologias

Inicialmente são necessárias ferramentas básicas para escrita e versionamento.

 - VS Code
 - Git

## Instalação

1. Faça o download do projeto ou clone o repositório
2. Conecte ao seu github
3. Crie uma branch com seu nome e outro dado que o identifique
4. Adapte para sua realidade como descrito a seguir

## Execução

O projeto está organizado em etapas, da mais fácil para a mais difícil.
Cada etapa tem um conjunto de conhecimentos a serem adquiridos e necessários para a etapa seguinte.
Para organizar, apoiar e controlar o estudo foram criados fluxos, do macro ao micro.
Apesar de ser um projeto pessoal, foi pensado de modo que outras pessoas possam adapta-lo para sua realidade.
Recomenda-se inicialmente uma leitura breve dos arquivos na seguinte sequência: Fluxo Macro, Fluxo Diário, Fluxo do Caderno.
